后端上传图片压缩规则（传给前端的缩略图）

1  后端上传图片压缩规则：

  1）如果上传的  高 或 宽 超过640，则需要压缩原图，压缩规则如下：

     系数 X = MIN（640/原图宽,640/原图高），保留4位小数

     则后端传给前端的缩略图压缩成

     宽：(原图宽 * X )

     高：(原图高 * X )  

     小数点处理方式：舍弃小数位取整

举例： 原图宽1280、原图高640，则系数=0.5，则压缩后宽=640、高=320


  2）如果上传的  高 和 宽 均未超过640，则不用压缩

2 原图 和 缩略图 都要保存到文件服务器。


附：前端的显示需求，后端了解即可：

1  前端展示图片拿后端的传来的缩略图根据手机宽度等比展示，

比如： 假设手机屏幕宽度 为375

则 前端显示图片宽固定 = 375

显示图片高的压缩系数 X = 375 / 后端传来的缩略图的宽

前端显示图片高 = 后端传来的缩略图的高 * X
